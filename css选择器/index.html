      # CSS选择器

      # 选择器命名规范

      1.可以包含数字、字母、下划线_、连字符-
2.不可以使用数字开头
3.不可以使用中文命名
4.不推荐使用我们的标签名或关键字命名
5.推荐使用驼峰式命名法 PiQiu
6.严格区分大小写

      # 选择器优先级（权重）

      样式 权重
      ！important（属性：值 !important;） 10000不推荐使用
      行内样式 01000
      id选择器 00100
      class/类选择器、伪类选择器 00010
      标签/元素选择器 00001
      全局/通配选择器 00000
      继承的样式没有权值 
      // 优先级规范
基础选择器CSS优先级——行内样式>id选择器>类选择器>标签选择器>全局选择器
后代选择器——选择器的权重值相加再比较

      # 基础选择器

      # （1）标签选择器/元素选择器

      通过标签名字（如p、div等）进行选择，选中的页面中指定的所有元素

      标签名{
  属性：值；
  属性：值；
}
      1
      2
      3
      4

      # （2）id选择器

      通过标签的id属性进行选择，选中页面中唯一一个元素

      具有唯一性，尽量少用

      #id属性值{
    属性：值；
    属性：值；
}
      1
      2
      3
      4

      # （3）class选择器/类选择器

      通过标签的class属性进行选择，选中页面中一类元素

      • 一类可以包含一个或者多个元素
      • 属性值可以是多个
      .class属性值{
  属性：值；
  属性：值；
}

——.one{

      去掉默认样式

      # （5）伪类选择器

      # （6）属性选择器

      标签名[属性]{
  属性：值；
  属性：值；
}
---[title]{}
---[title=""]{}
---p[titile]{}		选中带有title属性的p元素
---p[titile=""]{}	选中带有title属性指定值的p元素

---p[titile*=""]{}	选中带有title属性指定值包含的p元素
---p[titile^=""]{} 	选中带有title属性指定值开头的p元素
---p[titile$=""]{}  选中带有title属性指定值结尾的p元素

      # 复合选择器

      # （1）后代选择器

      选择作为某标签后代的标签。子集/包含

      • 选择器之间使用空格隔开
      • 选择器可以是任意类型的选择器
      选择器1 选择器2 选择器3{
  属性：值；
  属性：值；
}
      1
      2
      3
      4

      # （2）群组选择器/并集选择器

      作用：多个选择器实现同一样式时使用。

      语法：选择器，选择器，选择器{}

      注意：选择器是任意类型，选择器之间使用逗号隔开

      选择器，选择器，选择器{
  属性：值；
  属性：值；
}
      1
      2
      3
      4

      # （3）子选择器

      作用：查找某一标签的子级（儿辈）

      语法：选择器>选择器{}

      注意：选择器和选择器用大于号 隔开 选择器可以是任意类型

      选择器>选择器>选择器{
  属性：值；
  属性：值；
}
      1
      2
      3
      4

      # （4）交集选择器

      作用：交集选择器就是在两个标签相交的部分，也就是交集进行修改格式。

      对于交集选择器，可以与id和class共同使用。

      标签名.类名{
  属性：值；
  属性：值；
}

      2
      3
      4

      # （5）兄弟选择器

      相邻的第一个兄弟（选择器1后面第一个的选择器2）

      选择器+选择器{
  属性：值；
}
----两个选择器必须相邻

      # 伪类和伪元素

      # （1）概念

      CSS引入伪类和伪元素概念是为了格式化文档树以外的信息。也就是说伪类和伪元素是用来修饰不在文档树中的部分，比如，一句话中的第一个字母，或者是列表中的第一个元素。

---伪类用来当已有元素处于的某个状态时，为其添加对应的样式，这个状态是根据用户行为而动态变化的。
---伪元素用于创建一些不在文档树中的元素，并为其添加样式

      ：：first-line		---第一行

：：before...			---在...之前

      :link: 选择未访问的链接
:hover：选择鼠标指针浮动在其上的元素
