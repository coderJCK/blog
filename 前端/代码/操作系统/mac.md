# homebrew

## 安装
1. 官方安装方式
```linux
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
2. 镜像加速安装方式
```linux
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
```

## 常用命令
```linux
brew update
brew search [name]
brew install [name]
brew update [name]
brew info [name]
brew list
```


## 命令

~/.bash_profile 登录用户的环境变量设置
~/.zshrc 终端命令工具的环境变量设置
