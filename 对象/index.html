# 对象

// 字面量创建
var obj = {
    a:"123",
    b:[],
    c:function (){
        console.log(this.a)
    }
}
// new Object()创建
var obj = new Object();
obj.a = 12;
obj.b = "kk";
obj.c = function(){
    console.log(this.a)
}
// 工厂函数创建
function person(name,age){
    var obj = new Object();
    obj.a = name;
    obj.b = age;
    return person;
}
var obj = person("cc",22)

# 对象的静态方法

// Object.assign(目标对象,源对象1,源对象2)
---将所有可枚举属性的值从一个或多个源对象分配到目标对象
---返回目标对象
注：存在相同的属性时，源对象2会覆盖源对象1。
// Object.create()
---方法创建一个新对象，使用现有的对象来提供新创建的对象的__proto__
---现有对象是什么类型，返回什么类型
// Object.defineProperty(obj，"属性"，{})
---直接在一个对象上定义一个新属性，或者修改一个对象的现有属性
---返回此对象。
---set,get不能与其他属性同时使用 
注：{}内可以跟的东西：
(1)value：值
(2)writable：true/false		是否可以修改值
(3)configurable：true/false	 是否可以被删除
(4)enumerable：true/false	是否可以被枚举
(5)set
(6)get
// Object.entries()
---返回一个给定对象自身可枚举属性的键值对数组，其排列与使用 for...in 循环遍历该对象时返回的顺序一致（区别在于 for-in 循环还会枚举原型链中的属性）
// Object.freeze()
---冻结对象：其他代码不能删除或更改任何属性。
// Object.keys()
---返回一个包含所有给定对象自身可枚举属性名称的数组。
// Object.values()
---返回给定对象自身可枚举值的数组。
